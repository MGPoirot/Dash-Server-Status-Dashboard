TOP-LEVEL JSON STRUCTURE:

The root object MAY ONLY contain the following keys.

REQUIRED keys (MUST be present):
- "label"

- "type"
- "component"

- "display"

OPTIONAL keys (MAY be present):
- "description"

FIELD DEFINITIONS AND CONSTRAINTS:

1. "label"
- REQUIRED
- Type: string
- Short, human-readable title for the UI.

2. "description"
- OPTIONAL
- Type: string
- Human-readable description. HTML is allowed but not required.

3. "metric_id"
- REQUIRED
- Type: string
- Format: "[type]_[component]_[property]"
- Must be lowercase.
- Must match the values used in "type", "component", and "property".

4. "type"
- REQUIRED
- Type: string
- Category of the metric.
- Use a short lowercase noun (e.g. "system", "storage", "services", "script").

5. "component"
- REQUIRED
- Type: string
- Resource identifier (service name, disk name, component, etc.).
- All lowercase, hyphens are allowed
- DO NOT use spaces and underscores

6. "property"
- REQUIRED
- Type: string
- The measured aspect of the component (e.g. "up", "temp", "used-pct", "days").
- All lowercase, hyphens are allowed
- DO NOT use spaces and underscores

7. "schedule"
- REQUIRED
- Type: string
- How frequent the script this script should be run formatted as cron schedule expression. 
- MUST be exactly one of:
  - "weekly"
  - "bi-daily"
  - "daily"
  - "twice-daily"
  - "hourly"
  - "half-hourly"
  - "quarter-hourly"
  - "five-minutely"
  - "minutely"

8. "display"
- REQUIRED
- Type: object
- MAY ONLY contain the following keys:
  - "visual"
  - "tile_span"
  - "charts"

8a. "display.visual"
- REQUIRED
- Type: object
- MUST contain:
  - "type": string (required)
- "type" MUST be one of:
  - "gauge"
  - "number"
  - "counter"
  - "state"
  - "version"
  - "text"
- These are the parameters for each of the "display.visual.type" options:
  - when "display.visual.type" == "gauge":
    - "display.visual.min" REQUIRED as number
    - "display.visual.max" REQUIRED as number
    - "display.visual.hideAlerts" OPTIONAL as boolean
    - "display.visual.invert_y" OPTIONAL as boolean
  - when "display.visual.type" == "number":
    - "display.visual.min" OPTIONAL as number
    - "display.visual.max" OPTIONAL as number
    - "display.visual.hideAlerts" OPTIONAL as boolean
    - "display.visual.invert_y" OPTIONAL as boolean
    - "display.visual.nLatestPoints" OPTIONAL as number, the number of points that are relevant to show in the chart visual.
  - when "display.visual.type" == "counter": 
    - "display.visual.hideAlerts" OPTIONAL as boolean
    - "display.visual.nLatestPoints" OPTIONAL as number, the number of points back time over which the cumulative sum is calculated.
  - when "display.visual.type" == "version": 
    - "display.visual.nLatestPoints" OPTIONAL as number, the number of latest versions to show in the visual

8b. "display.tile_span"
- OPTIONAL
- Type: number
- Typical value: 1

8c. "display.charts"
- OPTIONAL
- Type: array of strings
- Each entry MUST be one of:
  - "line"
  - "area"
  - "bar"
  - "pie"

  - "priority": string, MUST be one of "info", "warning", "critical"
